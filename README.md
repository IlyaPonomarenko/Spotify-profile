## Spotify Profile with TypeScript & REACT

With this app built with React and TypeScipt you can have a look at your top tracks and artists, see currently playing track and 10 recent tracks listened. 

### Technologies Used

- React
  - useContext
- TypeScript
- Spotify API
- HTML
- CSS
